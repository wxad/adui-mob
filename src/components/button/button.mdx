# Button

用于承载点击触发行为的控件，如跳转、选择等。

### 常用示例

#### 基本用法

```javascript
// 主题按钮
<ad-button>主题按钮</ad-button>
// 成功按钮
<ad-button intent="succes">成功按钮</ad-button>
// 警告
<ad-button intent="warning">警告按钮</ad-button>
// 危险按钮
<ad-button intent="danger">危险按钮</ad-button>
// 加载态
<ad-button loading>加载中</ad-button>
// 禁用态
<ad-button disabled>禁用</ad-button>
```

#### 不同风格

```javascript
// 默认风格
<ad-button>默认风格</ad-button>

// 轻量风格
<ad-button theme="light">轻量风格</ad-button>

// 文字按钮风格
<ad-button theme="text">文字风格</ad-button>

```

#### 含图标

```javascript
// 左图标
<ad-button leftIcon="add">左图标</ad-button>

// 右图标
<ad-button rightIcon="add">右图标</ad-button>
```

#### 不同尺寸

button 有两种尺寸，可选值为 medium 和 small，默认是 medium，如果想让 button 撑满父节点，可以通过配置 banner 属性实现
```javascript
// 默认尺寸
<ad-button>默认尺寸</ad-button>

// 小尺寸
<ad-button size="small">小尺寸</ad-button>

// 撑满父节点
<ad-button banner>撑满父节点</ad-button>
```
