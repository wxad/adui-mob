# Picture

支持多种填充模式，且包含懒加载、加载中提示、加载失败提示等状态的控件。


### 常用示例

#### 填充方式

picture 组件支持小程序原生组件 image 组件的所有 mode

```javascript
<ad-picture
  style="width: 100%; height: 100%;"
  mode="aspectFill"
  src="https://wxa.wxs.qq.com/images/paki/adui-for-mobile/coffe.png"
/>
```

<Alert title="注意事项" text="为了使组件使用更加灵活，picture 默认是没有尺寸大小的，尺寸需要在外部自行定义"/>

#### 点击预览

```javascript
<ad-picture
  style="width: 100%; height: 100%;"
  mode="aspectFill"
  previewable
  src="https://wxa.wxs.qq.com/images/paki/adui-for-mobile/coffe.png"
/>
```

#### 占位图

```javascript
  // 自定义占位的 icon，icon 的可选值为 icon 组件的值
  <ad-picture placeholderIcon="image" src="" />
```
