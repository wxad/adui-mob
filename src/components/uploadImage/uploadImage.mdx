# UploadImage

上传组件可以帮助用户对上传过程和上传结果有预期，并可以更改或撤销上传行为。

### 常用示例

#### 基本用法

```javascript
  <ad-upload-image />
  // 带描述
  <ad-upload-image desc="上传图片" />
```

#### 底纹样式

UploadImage 组件提供了部门底纹样式可以直接使用，有 IDPerson（身份证人像面）、IDNation（身份证国徽面）、businessLicense（营业执照）、serviceLicense（互联网新闻信息服务许可证）四种类型

```javascript
// 身份证
<view class="block" style="display: flex;">
  <ad-upload-image
    style="display: block; flex: 1; margin-right: 10px;"
    desc="人像面"
    placeholderType="IDPerson"
    banner="{{false}}"
  />
  <ad-upload-image
    desc="国徽面"
    style="flex: 1"
    banner="{{false}}"
    placeholderType="IDNation"
  />
</view>

// 营业执照
<ad-upload-image desc="营业执照" placeholderType="businessLicense" />

// 服务许可证

<ad-upload-image desc="互联网新闻信息服务许可证" placeholderType="serviceLicense" />
```
