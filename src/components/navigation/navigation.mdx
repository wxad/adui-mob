# Navigation

用于自定义页面的头部，一般情况下建议使用原生的头部，当需要自定义头部的时候才使用

### 常用示例

#### 基本用法

```javascript
<ad-navigation title="页面标题" icon="back" background="#fff" color="#000000" />
```

#### 固定在顶部

```javascript
<ad-navigation title="页面标题" icon="back" background="#fff" color="#000000" isFixed />
```

#### 不同的icon

```javascript
<ad-navigation title="页面标题" icon="" background="#fff" color="#000000"  isFixed />
<ad-navigation title="页面标题" icon="back" background="#fff" color="#000000"  isFixed />
<ad-navigation title="页面标题" icon="close" background="#fff" color="#000000"  isFixed />
<ad-navigation title="页面标题" icon="home" background="#fff" color="#000000"  isFixed />
```
