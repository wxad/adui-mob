# Tag

用于标记、分类和选择的控件。

### 常用示例

#### 基本用法

```javascript
const intents = [
  { name: '主题', intent: 'primary' },
  { name: '成功', intent: 'success' },
  { name: '警告', intent: 'warning' },
  { name: '危险', intent: 'danger' },
  { name: '自定义', intent: 'normal' },
]

<ad-tag wx:for="{{intents}}" intent="{{item.intent}}"> {{item.name}} </ad-tag>

```

#### 不同风格

tag 有四种风格，分别为 standard、light、outlined、text，分别对应固态、轻量、空心样式和文字样式，默认是 light。

```javascript

<ad-tag theme="light">轻量风格</ad-tag>
<ad-tag theme="standard">固态风格</ad-tag>
<ad-tag theme="outlined">空心风格</ad-tag>
<ad-tag theme="text">文字风格</ad-tag>

```

#### 组合图标和禁用态

```javascript
<ad-tag theme="standard" intent="primary" leftIcon="happy"> 左图标 </ad-tag>
<ad-tag theme="standard" intent="primary" leftIcon="tick-circle-outlined"> 右图标 </ad-tag>
<ad-tag theme="standard" intent="primary" disabled> 禁用 </ad-tag>
```
