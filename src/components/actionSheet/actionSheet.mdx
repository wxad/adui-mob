# ActionSheet

从底部弹出，提供当前场景相关的 2 个以上的操作动作。主要采用文字按钮，重要敏感的操作一般会用主题颜色或红色显示。

### 常用示例

#### 基础用法

```javascript
<ad-action-sheet
  visible="{{isActionSheetVisible}}"
  bind:onChange="close"
  bind:onClose="close"
>
  <ad-action-sheet-item title="选项一" value="{{1}}" />
  <ad-action-sheet-item title="选项二" value="{{2}}" />
  <ad-action-sheet-item title="选项三" value="{{3}}" />
</ad-action-sheet>
```

#### 展示取消按钮

```javascript
<ad-action-sheet
  visible="{{isActionSheetVisible}}"
  bind:onChange="close"
  bind:onCancel="close"
  bind:onClose="close"
  hasCancel
>
  <ad-action-sheet-item title="选项一" value="{{1}}" />
  <ad-action-sheet-item title="选项二" value="{{2}}" />
  <ad-action-sheet-item title="选项三" value="{{3}}" />
</ad-action-sheet>
```

#### 带描述信息

```javascript
<ad-action-sheet
  visible="{{isActionSheetVisible}}"
  bind:onChange="close"
  bind:onCancel="close"
  bind:onClose="close"
  hasCancel
  desc="这是一段描述"
>
  <ad-action-sheet-item title="选项一" desc="描述信息" value="{{1}}" />
  <ad-action-sheet-item title="选项二" desc="描述信息" value="{{2}}" />
</ad-action-sheet>
```

#### 选项带状态

```javascript
<ad-action-sheet
  visible="{{showDrawer4}}"
  bind:onChange="close"
  bind:onCancel="close"
  bind:onClose="close"
  hasCancel
>
  <ad-action-sheet-item
    title="着色选项"
    customStyle="color: var(--primary-color)"
    value="{{1}}"
  />
  <ad-action-sheet-item title="选项二" disabled value="{{2}}" />
  <ad-action-sheet-item
    title="选项二"
    loading
    loadingText="加载中..."
    value="{{3}}"
  />
</ad-action-sheet>
```
