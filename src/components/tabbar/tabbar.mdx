# Tabbar

用于自定义页面的头部，一般情况下建议使用原生的头部，当需要自定义头部的时候才使用

<Alert
  title="组件间关系"
  text="TabbarItem 组件的祖先组件，需要搭配 TabbarItem 使用"
/>

### 常用示例

#### 基本用法

```javascript
<ad-tabbar class="item" defaultValue="{{0}}">
  <ad-tabbar-item value="{{0}}">
    <image class="icon" slot="icon" src="../images/home.svg" />
    <image class="icon" slot="icon-active" src="../images/home_selected.svg" />
    首页
  </ad-tabbar-item>
  <ad-tabbar-item value="{{1}}">
    <image class="icon" slot="icon" src="../images/manage.svg" />
    <image class="icon" slot="icon-active" src="../images/manage_selected.svg" />
    管理
  </ad-tabbar-item>
</ad-tabbar>
```

#### 组合徽标

```javascript
<ad-tabbar class="item" defaultValue="{{0}}">
  <ad-tabbar-item value="{{0}}">
    <image class="icon" slot="icon" src="../images/home.svg" />
    <image class="icon" slot="icon-active" src="../images/home_selected.svg" />
    首页
  </ad-tabbar-item>
  <ad-tabbar-item value="{{2}}" badgeContent="">
    <image class="icon" slot="icon" src="../images/message.svg" />
    <image class="icon" slot="icon-active" src="../images/message_selected.svg"/>
    消息
  </ad-tabbar-item>
</ad-tabbar>
```
