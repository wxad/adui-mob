# CellGroup

单个连续模块垂直排列，显示当前内容、状态和可进行对操作。

<Alert
  title="组件间关系"
  text="cell 组件的祖先组件，使用的时候搭配 cell 组件"
/>

### 常用示例

#### 基本用法

```javascript
<ad-cell-group>
  <ad-cell title="标题" />
  <ad-cell title="标题" placeholder="描述文字右对齐" />
  <ad-cell
    title="标题"
    placeholder="描述文字左对齐"
    rightStyle="justify-content: flex-start;"
  />
</ad-cell-group>
```

#### 双标题

```javascript
<ad-cell-group>
  <ad-cell title="标题" desc="副标题尽量保持一行" />
  <ad-cell title="标题" desc="副标题尽量保持一行" rightDesc="10:15" />
</ad-cell-group>
```

#### 带图片或者图标

```javascript
<ad-cell-group>
  <ad-cell
    title="标题"
    avatar="https://wxa.wxs.qq.com/images/paki/adui-for-mobile/avatar.png"
  />
  <ad-cell
    title="小程序助手"
    leftIcon="mini-program"
    leftIconColor="#7587DB"
    isLink
  />
</ad-cell-group>
```

#### 可操作

一般情况下 cell 会和其他组件一起搭配使用，这时可以使用 slot 来进行组合

```javascript
<ad-cell-group>
  // 配置 isLink 属性后 cell 会带上跳转箭头，具有点击交互性
  <ad-cell title="标题" isLink />
  // 组合 switch 组件
  <ad-cell title="标题">
    <ad-switch slot="rightElement" defaultChecked></ad-switch>
  </ad-cell>
  // 组合步进器组件
  <ad-cell title="标题">
    <ad-stepper slot="rightElement" value="{{30}}" />
  </ad-cell>
</ad-cell-group>
```

#### 分割线

默认情况下 cell 的分割线会在左右两端留出 20px 的空白，可以设置只有右边通栏和全部通栏

```javascript
// 默认左右留出 20px 的空白
<ad-cell-group>
  <ad-cell title="默认" isLink />
  <ad-cell title="默认" isLink />
</ad-cell-group>

// 右通栏
<ad-cell-group>
  <ad-cell
    rightBanner
    title="右通栏"
    isLink
  />
  <ad-cell
    rightBanner
    title="右通栏"
    isLink
  />
</ad-cell-group>

// 左右通栏
<ad-cell-group>
  <ad-cell
    banner
    title="通栏"
    isLink
  />
  <ad-cell
    banner
    title="通栏"
    isLink
  />
</ad-cell-group>
```
