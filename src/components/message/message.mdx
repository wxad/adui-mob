# Message

承载系统告示、提醒等反馈行内容。

### 常用示例

#### 基础样式

```javascript
const list = [
  { title: '成功类消息', intent: 'success' },
  { title: '提醒类消息', intent: 'primary' },
  { title: '警告类消息', intent: 'warning' },
  { title: '危险类消息', intent: 'danger' },
  { title: '告示类消息', intent: 'normal' },
],

<ad-message
  class="message"
  title="{{item.title}}"
  intent="{{item.intent}}"
  wx:for="{{list}}"
/>
```

#### 带描述说明

```javascript
<ad-message
  class="message"
  title="成功类消息"
  intent="success"
  desc="此处是描述内容"
/>
```

#### 带图标

```javascript
<ad-message
  class="message"
  title="成功类消息"
  intent="success"
  hasIcon
  desc="此处是描述内容"
/>
```

#### 卡片式

```javascript
<ad-message
  class="message"
  title="成功类消息"
  intent="success"
  isCard
  desc="此处是描述内容"
/>
```

#### 可跳转

```javascript
<ad-message
  class="message"
  title="成功类消息"
  intent="success"
  hasIcon
  isLink
/>
```
