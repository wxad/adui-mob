# Toast

一种轻量级的反馈/提示，可以用来显示不会打断用户操作的内容，适用于页面转场、数据交互等场景中。

### 常用示例

#### 基本用法

一般情况下可以通过 visible 来控制 toast 的展示和隐藏。需要注意的是一般情况下 toast 展示后会在设置的 duration 时间后自动隐藏，但是如果是 loading 态的 toast 则需要手动设置 toast 的隐藏
```javascript
  <ad-toast visible="{{isToastVisible}}" title="文案超过10字就折行" />
  <ad-toast visible="{{isToastVisible}}" loading title="文案超过10字就折行" />
```
