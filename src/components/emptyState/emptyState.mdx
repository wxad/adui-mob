# EmptyState

当内容区域（页面、区块、组件、数据）没有内容显示给用户时，就会出现空状态。

### 常用示例

#### 基本样式

```javascript
  <ad-empty-state text="搜索结果为空" type="search" />
  <ad-empty-state text="没有找到门店" type="store" />
  <ad-empty-state text="暂无广告列表" type="list" />
  <ad-empty-state text="暂无筛选项" type="filter" />
  <ad-empty-state text="暂无新通知" type="news" />
```
