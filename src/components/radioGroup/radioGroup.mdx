# RadioGroup

支持同时选中单项的基础控件。

<Alert title="组件间关系" text="radio 组件的祖先组件，需要搭配 radio 使用" />

### 常用示例

radio 和 radioGroup 组件一般是搭配 cell 和 cellGroup 一起使用，通过外部控制来控制值的变化，祖孙组件之间可以互相嵌套，可以使用 radioField 组件来自定义触发区域

#### 基本用法

```javascript
<ad-radio-group value="选中">
  <ad-cell-group noTopBorder noBottomBorder>
    <ad-radio-field>
      <ad-cell interactive>
        <ad-radio slot="leftElement" value="常态">
          常态
        </ad-radio>
      </ad-cell>
    </ad-radio-field>
    <ad-radio-field>
      <ad-cell interactive>
        <ad-radio slot="leftElement" value="选中">
          选中
        </ad-radio>
      </ad-cell>
    </ad-radio-field>
  </ad-cell-group>
</ad-radio-group>
```

#### 禁用态

radioGroup 的 disabled 可以禁止整个 radioGroup 组件，如果只想要某个 radio 组件禁用，可以通过 radio 本身的 disabled 来进行配置

```javascript
<ad-radio-group defaultValue="选中" disabled>
  <ad-cell-group noBottomBorder>
    <ad-radio-field>
      <ad-cell>
        <ad-radio slot="leftElement" value="常态">
          禁用
        </ad-radio>
      </ad-cell>
    </ad-radio-field>
    <ad-radio-field>
      <ad-cell>
        <ad-radio slot="leftElement" value="选中">
          选中并禁用
        </ad-radio>
      </ad-cell>
    </ad-radio-field>
  </ad-cell-group>
</ad-radio-group>
```

#### 轻量样式

```javascript
<ad-radio-group value="选中">
  <ad-cell-group noTopBorder noBottomBorder>
    <ad-radio-field>
      <ad-cell interactive>
        <ad-radio slot="leftElement" value="常态" theme="light">
          常态
        </ad-radio>
      </ad-cell>
    </ad-radio-field>
    <ad-radio-field>
      <ad-cell interactive>
        <ad-radio slot="leftElement" value="选中" theme="light">
          选中
        </ad-radio>
      </ad-cell>
    </ad-radio-field>
  </ad-cell-group>
</ad-radio-group>
```
