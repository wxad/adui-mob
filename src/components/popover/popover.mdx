# Popover

用户点击某个功能后浮出临时气泡对其作出补充，更强调功能的归属，可以出现在页面的任何地方。气泡通常会伴随着半透明的遮罩或者投影衬底。

### 常用示例

#### 基础样式

```javascript
<ad-popover
  placement="topLeft"
  class="item"
  visible="{{true}}"
  bind:onClose="closePop"
>
  <ad-button bind:tap="openPop" >{{item.title}}</ad-button>
  <view slot="pop">哈哈哈哈哈哈哈哈哈哈或</view>
</ad-popover>
```
<Alert
  text="popover 在使用的时候必须同时使用默认的 slot 和 pop slot 来承载 trigger 和 弹出层内容，同时在 popover 展示的时候会有一层透明的蒙层盖在最上面来防止页面的交互，也就是说只有当 popover 隐藏后，页面才是可交互的"
  title="注意事项"  
/>
