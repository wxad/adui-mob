# Tab

切换导航至同一层级的内容页面。

### 常用示例

#### 基本用法

```javascript

const tabs = [
    { title: '标签一', value: 0 },
    { title: '标签二', value: 1 },
]


<ad-tabs bind:onChange="tabChange" tabItems="{{tabs}}" />
```

#### 轻量样式

```javascript
const tabs = [
    { title: '标签一', value: 0 },
    { title: '标签二', value: 1 },
]
<ad-tabs bind:onChange="tabChange" tabItems="{{tabs}}"  theme="light"/>
```

#### 禁用态

```javascript
const tabs = [
    { title: '标签一', value: 0 },
    { title: '禁用态', value: 2, disabled: true },
]

<ad-tabs bind:onChange="tabChange" tabItems="{{tabs}}"  theme="light"/>
```
