# Badge

出现在按钮、图标旁的数字或状态标记。

### 常用示例

#### 基本用法

```javascript
// 红点
<ad-badge isDot/>
// 填充数值
<ad-badge >99+</ad-badge>
// 填充任意文案
<ad-badge isFixed contentStyle="background-color: #07C160; font-weight: 700; font-size: 8px; line-height: 8px; letter-spacing: 0.1em;">NEW</ad-badge>
```


#### 组合使用

一般 badge 都会和其他组件组合在一起使用，如下几个示例

```javascript
// 组合头像
<view style="position: relative;">
  <ad-avatar
    size="medium"
    src="https://wxa.wxs.qq.com/images/paki/adui-for-mobile/picture.png"
  />
  <ad-badge isFixed>1</ad-badge>
</view>
// 组合 icon
<view style="position: relative;">
  <ad-icon
    slot="leftElement"
    icon="image-outlined"
    customStyle="position: relative"
  />
  <ad-badge isFixed>∙∙∙</ad-badge>
</view>

// 组合 cell
<ad-cell title="列表" isLink>
  <ad-badge slot="rightElement" isDot/>
</ad-cell>
```
