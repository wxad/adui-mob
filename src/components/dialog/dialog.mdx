# Dialog

用作显示系统的重要信息，并请求用户进行操作反馈，eg：删除某个重要内容时，弹出 Modal 进行二次确认。

### 常用示例

#### 基础样式

只有确定按钮

```javascript
<ad-dialog
  visible="{{true}}"
  title="欢迎"
  type="inform"
  bind:onConfirm="close"
  bind:onCancel="close"
>
  <view>欢迎使用 AD UI 小程序</view>
</ad-dialog>
```

确定和取消按钮

```javascript
// 确定和取消按钮
<ad-dialog
  visible="{{true}}"
  title="欢迎"
  bind:onConfirm="close"
  bind:onCancel="close"
>
  <view>欢迎使用 AD UI 小程序</view>
</ad-dialog>
```

多个按钮

```javascript
  const buttonList = [
    { value: 0, text: '选项一' },
    { value: 1, text: '选项二' },
    { value: 2, text: '选项三' },
  ]
  <ad-dialog
    visible="{{true}}"
    title="欢迎"
    bind:onConfirm="close"
    bind:onCancel="close"
    buttonList="{{buttonList}}"
  >
    <view>
      欢迎使用 AD UI 小程序
    </view>
  </ad-dialog>
```
