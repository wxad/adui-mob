# TabbarItem

用于自定义页面的头部，一般情况下建议使用原生的头部，当需要自定义头部的时候才使用

<Alert
  title="组件间关系"
  text="Tabbar 组件的子孙组件，需要搭配 Tabbar 使用，使用是需要配置 value 作为每个 TabbarItem 的唯一标识"
/>

export const slots = [

  {
    name: 'iconActive',
