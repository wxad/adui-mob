# Spinner

用于告知用户信息内容的加载状态，通常是即时、短时的。

### 常用示例

#### 基本用法

```javascript
<ad-spinner />
```

#### 不同尺寸

```javascript
  <ad-spinner size="large" />
  <ad-spinner size="medium" />
  <ad-spinner size="small" />
```

#### 带加载文案

```javascript
  // 水平排列
  <ad-spinner text="加载中..." />
  // 垂直排列
  <ad-spinner text="加载中..." direction="vertical" />
```

#### 自定义颜色

```javascript
  // 主题颜色
  <ad-spinner />
  // 黑色
  <ad-spinner color="#1f1f1f"/>
  // 白色
  <ad-spinner color="#fff" />
```
