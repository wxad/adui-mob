# Stepper

步进器由增加按钮、减少按钮和输入框组成，用于在一定范围内输入、调整数字。

### 常用示例

#### 基本用法

```javascript
<ad-stepper value="{{20}}" />
// 步长设置（整数）
<ad-stepper value="{{20}}" step="{{10}}"/>
// 步长设置（小数）
<ad-stepper value="{{20}}" step="{{0.3}}"/>
// 限制输入范围(左)
<ad-stepper value="{{20}}" min="{{20}}"/>
// 限制输入范围(右)
<ad-stepper value="{{20}}" max="{{20}}"/>
```

#### 状态

```javascript
// 输入值超出限制
<ad-stepper value="{{20}}" min="{{30}}" max="{{60}}" bind:onError="error"/>
// 禁用态
<ad-stepper disabled value="{{20}}"/>
```
