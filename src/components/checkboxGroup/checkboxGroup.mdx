# CheckboxGroup

支持同时选中多项的基础控件。

<Alert
  title="组件间关系"
  text="checkbox 组件的祖先组件，需要搭配 checkbox 使用"
/>

### 常用示例

#### 基本用法

checkbox 和 checkboxGroup 组件一般是搭配 cell 和 cellGroup 一起使用，通过外部控制来控制值的变化，祖孙组件之间可以互相嵌套，可以使用 checkboxField 来自定义触发的区域

```javascript
<ad-checkbox-group value="{{['选中']}}" bind:onChange="change">
  <ad-cell-group>
    <ad-checkbox-field>
      <ad-cell interactive>
        <ad-checkbox value="常态" slot="leftElement">
          常态
        </ad-checkbox>
      </ad-cell>
    </ad-checkbox-field>
    <ad-checkbox-field>
      <ad-cell interactive>
        <ad-checkbox value="选中" slot="leftElement">
          选中
        </ad-checkbox>
      </ad-cell>
    </ad-checkbox-field>
  </ad-cell-group>
</ad-checkbox-group>
```

#### 禁用态

checkboxGroup 的 disabled 可以禁止整个 checkboxGroup 组件，如果只想要某个 checkbox 组件禁用，可以通过 checkbox 本身的 disabled 来进行配置

```javascript
// 整个禁用
<ad-checkbox-group defaultValue="{{['选中']}}" disabled>
  <ad-cell-group>
    <ad-checkbox-field>
      <ad-cell>
        <ad-checkbox value="常态" slot="leftElement">
          禁用
        </ad-checkbox>
      </ad-cell>
    </ad-checkbox-field>
    <ad-checkbox-field>
      <ad-cell>
        <ad-checkbox value="选中" slot="leftElement">
          选中并禁用
        </ad-checkbox>
      </ad-cell>
    </ad-checkbox-field>
  </ad-cell-group>
</ad-checkbox-group>

// 部分禁用
<ad-checkbox-group defaultValue="{{['选中']}}">
  <ad-cell-group>
    <ad-checkbox-field>
      <ad-cell>
        <ad-checkbox value="常态" slot="leftElement">
          可用
        </ad-checkbox>
      </ad-cell>
    </ad-checkbox-field>
    <ad-checkbox-field>
      <ad-cell>
        <ad-checkbox value="选中" slot="leftElement" disabled>
          选中并禁用
        </ad-checkbox>
      </ad-cell>
    </ad-checkbox-field>
  </ad-cell-group>
</ad-checkbox-group>
```

#### 带描述

```javascript
<ad-checkbox-group defaultValue="{{['选中']}}">
  <ad-cell-group>
    <ad-checkbox-field>
      <ad-cell>
        <ad-checkbox value="常态" slot="leftElement" desc="描述文字">
          常态
        </ad-checkbox>
      </ad-cell>
    </ad-checkbox-field>
    <ad-checkbox-field>
      <ad-cell>
        <ad-checkbox value="选中" slot="leftElement" desc="描述文字">
          选中
        </ad-checkbox>
      </ad-cell>
    </ad-checkbox-field>
  </ad-cell-group>
</ad-checkbox-group>
```
