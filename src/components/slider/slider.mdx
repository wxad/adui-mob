# Slider

滑动条用于以拖拽的方式，更人性化地完成输入数据、选择数据区间等目的。

### 常用示例

#### 基础用法

基本样式

```javascript
<ad-slider min="{{0}}" max="{{100}}" defaultValue="{{50}}" />
```

带刻度标记

```javascript
const marks = [
  { position: 0, value: '0' },
  { position: 25, value: '25' },
  { position: 50, value: '50' },
  { position: 75, value: '75' },
  { position: 100, value: '100' },
]
<ad-slider
  min="{{0}}"
  max="{{100}}"
  marks="{{marks}}"
  defaultValue="{{50}}"
/>
```

带顶部提示

```javascript
<ad-slider
  class="slider"
  min="{{0}}"
  max="{{100}}"
  bind:onChange="change"
  defaultValue="{{value}}"
  hasTip
>
  <view slot="endTip">{{ value }}</view>
</ad-slider>
```

#### 双滑块模式

```javascript

const marks = [
  { position: 0, value: '0' },
  { position: 25, value: '25' },
  { position: 50, value: '50' },
  { position: 75, value: '75' },
  { position: 100, value: '100' },
]

<ad-slider
  class="slider"
  min="{{10}}"
  max="{{90}}"
  defaultValue="{{value}}"
  hasTip
  bind:onChange='change'
  range
  marks="{{marks}}"
>
  <view slot="startTip">{{value[0]}}</view>
  <view slot="endTip">{{value[1]}}</view>
</ad-slider>

```
