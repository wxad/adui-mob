# Select

用于承载较多当选择项内容，可展开与收起的组件。

<Alert
  text="selectItem 的祖先组件，用于承载 selectItem 的内容，需与 selectItem 一起使用"
  title="组件间关系"
/>

### 常用示例

#### 基础用法

```javascript
const options1 = [
  {
    title: '选项一',
    value: 0,
  },
  {
    title: '选项二',
    value: 1,
  },
  {
    title: '选项三',
    value: 2,
  },
  {
    title: '选项四',
    value: 3,
  },
]
<ad-select>
  <ad-select-item
    key="{{0}}"
    options="{{options1}}"
    defaultValue="{{value1}}"
  />
  <ad-select-item
    key="{{1}}"
    options="{{options1}}"
    defaultValue="{{value2}}"
  />
</ad-select>
```

<Alert
  text="当与 select 一起使用时，每个 selectItem 必须提供 key，这是每个 selectItem 的唯一标识；select 头部每个子项默认是 selectItem 的 value，如果 selectItem 没有设置 value，默认是第一个选项"
  title="注意事项"
/>

#### 状态

```javascript
<ad-select>
  <ad-select-item key="{{0}}" options="{{options1}}" defaultValue="{{0}}" />
  <ad-select-item
    key="{{1}}"
    options="{{options1}}"
    defaultValue="{{1}}"
    disabled
  />
</ad-select>
```
