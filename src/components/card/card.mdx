# Card

用于组织信息和操作的控件，也可作为信息入口。

### 常用示例

#### 基本用法

```javascript
<ad-card>
  <view style="height: 68px" />
</ad-card>
```

#### 通栏

```javascript
<ad-card banner>
  <view style="height: 68px" />
</ad-card>
```
