# Switch

用于开启或关闭选项功能的控件。

### 常用示例

#### 基本用法

```javascript
  // 默认开启
	<ad-switch defaultChecked />
   // 禁用
  <ad-switch disabled />
  // 开启并禁用
  <ad-switch checked disabled />
```
